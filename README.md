# 🔥 FitTransform 12 — 12-Week Fitness Transformation Planner

A complete **60-minute daily workout planner** built as a single HTML file. No app install needed — works in any browser and can be installed as a mobile app (PWA).

---

## ✨ Features

- **60-Min Integrated Sessions** — Cardio + Strength + Core combined every day
- **Push / Pull / HIIT / Active Rest** rotation across the week
- **3 Phases** — Foundation (W1–4) → Build (W5–8) → Shred (W9–12)
- **YouTube links** on every exercise for proper form tutorials
- **Diet Plan** — Vegetarian + Eggs, 6 meals/day with macros
- **Progress Tracker** — Weight, waist & belly measurements with trend charts
- **localStorage** — All progress saved locally in your browser
- **PWA Ready** — Install on Android/iOS like a native app
- **Light + Dark Mode** — Light by default, dark mode toggle
- **Fully Mobile Responsive** — Works on any screen size

## 📲 Install as Mobile App (PWA)

### Android (Chrome)
1. Open the GitHub Pages URL in **Chrome**
2. Tap the **three-dot menu** (⋮) → **Add to Home Screen**
3. Tap **Install** — it appears on your home screen like a native app

### iOS (Safari)
1. Open the URL in **Safari**
2. Tap the **Share** button → **Add to Home Screen**
3. Tap **Add** — done ✅

---

## 🛠️ Convert to APK (Android Package)

1. Open **[pwabuilder.com](https://pwabuilder.com)**
2. Paste your GitHub Pages URL
3. Click **Start** → **Package for stores** → **Android**
4. Download the APK
5. Transfer to your phone → Install (allow unknown sources in Settings)

## 🚀 How to Use

1. Open `index.html` in any browser (or visit the GitHub Pages URL)
2. Select your **Week** (W1–W12) and **Day** (Mon–Sun)
3. Follow the 60-min workout plan
4. Tap each exercise name to watch a **YouTube tutorial**
5. Check off exercises as you complete them
6. Mark the **day as complete** when done
7. Log your **measurements** weekly in the Progress tab
8. Track your transformation with the **trend charts**

## 💡 Key Tips for Side & Belly Fat

1. Hit **120–150g protein** daily — eggs, paneer, dal, Greek yogurt
2. **10–12 hour eating window** — last meal by 9 PM, breakfast by 7 AM
3. **3–4L water** per day — reduces bloating significantly
4. **Core every session** — planks, Russian twists, bicycle crunches
5. **Sleep 7–9 hours** — poor sleep = high cortisol = belly fat storage
